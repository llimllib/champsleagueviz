NBA Divisional Probability

Each team is placed here by the median probability, via oddschecker, that it will win its NBA Division.

The data from oddschecker is downloaded periodically and will be kept refreshed througout the season. If you want to see the code used to download the data and generate the chart, head over to github and check it out.
